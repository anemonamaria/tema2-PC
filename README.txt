
Funtia de Destroy in advance poate fi folosita prin apasarea tastelor 'e' sau 'E', iar cea de Randomize map prin tastarea 'R' sau 'r'.

Jucatorul va trebui sa treaca intr-un fisier harta sa prin marcarea barcilor cu 'x', iar a spatiilor libere cu ' ' intr-o tabela de 10x10 dupa cum este si tabela din fisierul input.txt.
